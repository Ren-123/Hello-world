数据库了解

常见数据库产品

  • Oracle: 甲骨文, 贵
  • DB2: IDM
  • SQL Server: 微软; 还可以, 兼容性有问题
  • MySQL, 甲骨文

数据库相关概念

  • DB(DataBase)

    数据库; 存储数据的仓库, 保存了一系列有组织的数据

  • DBMS(DB Management System)

    数据库管理系统.

    数据库是通过DBMS创建和操作的容器

  • SQL(Structure Query Language)

    结构化查询语言, 专门用来与数据库通信的语言

数据库存储数据的特点

  • 将数据放入表中, 在讲表放入库中
  • 一个数据库中可以有多个表, 每个表都有一个自己的名字, 用来表示自己, 具有唯一性
  • ORM

MySQL入门

MySQL数据库的简介

  • 开源 关系型数据库管理系统
  • 体积小, 速度快, 成本低

MySQL服务端安装和卸载

DBMS 分为两类:

  • 基于共享文件系统的DBMS(Access)
  • 基于客户机-服务器的DBMS(C/S)(MySQL, Oracle, SQLServer)

**卸载: **

  1. 先停止MySQL的服务(服务里停止)
  2. 点击卸载uninstall.exe
  3. 清除残余文件
    • MySQL安装目录下
    • C:\ProgramData\MySQL
  4. 清理注册表(略)

安装:

  • 安装后配置: MySQLInstanceConfig.exe
  • 配置文件修改: My.ini
    • 编码格式
    • 存储引擎 INNODB
    • 语法模式SQL-mode
    • 最大连接数max-connection

MySQL服务的启动和停止

  • 图形化界面

    服务->MySQL->启动

  • 命令行

    1. mysql是名称, 可能是mysql80等
    2. 要以管理员身份执行

MySQL服务的连接和退出

  • 方式1. MySQL自带客户端

    开始菜单 -> MySQL Server -> MySQL Command Line Client

    只能以root用户方式登录

  • 方式2. 终端中登录

    -p不能加空格, 其他的可以加
